Foundation

Information

Foundation bygger på att ge utvecklare möjligheten till att utveckla webbplatser smidigt med en mobile-first design. Ramverket är responsivt och gör det enkelt att designa webbplatser som fungerar på alla typer av skärmar, oavsett storlek. Foundation har både stöd för JavaScript och CSS och det går att inkludera via CDN, via command line (CLI), ladda ner och inkludera och via npm paket. För att kunna göra egna inställningar behöver man använda SASS för att ställa in variabler.

Fördelar och nackdelar

Foundation är väldigt lätt att installera och är inte särskillt stort så det går bra att använda exempelvis CDN-länkar om man vill det. Det är enkelt att designa en responsiv webbplats och enkelt att hitta lösningar på deras webbsida. Foundation har både egna klasser som man kan lägga till. Ett exempel på det är de klasser som används för att få bild bredbid text på det sättet som jag använder här.

Sen kan man exempelvis lägga till klassen "stack-for-small" för att sidan ska se bra ut på små skärmar. Dessutom innehåller Foundation CSS-regler för vissa HTML-element som är både bra och dåligt. Det är smidigt att inte behöver lägga till klasser men irriterande att behöva styla om element som man inte vill ha någon style på. Foundation är dessutom ganska simpelt och färra funktioner än de stora ramverken, som exempelvis botstrap.
