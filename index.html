CSS-ramverk

I den här uppgiften testar jag olika ramverk för CSS. Ramverk för CSS snabbar upp utvecklingen av ett nytt projekt och gör design smidigt och enkelt. Uppgiften går ut på att testa olika CSS-ramverk och utvärdera dem på en webbsida. Dessutom ska webbsidan laddas upp genom ett alternativt sätt (alltså ej via ftp) Jag har valt att använda Foundation och Tailwind CSS och att publicera webbplatsen via Github-pages. Guthub-pages använder git för att publicera en webbplats
