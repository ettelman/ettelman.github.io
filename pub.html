Github Pages

Foundation bygger på att ge utvecklare möjligheten till att utveckla webbplatser smidigt med en mobile-first design. Ramverket är responsivt och gör det enkelt att designa webbplatser som fungerar på alla typer av skärmar, oavsett storlek. Foundation har både stöd för JavaScript och CSS och det går att inkludera via CDN, via command line (CLI), ladda ner och inkludera och via npm paket. För att kunna göra egna inställningar behöver man använda SASS för att ställa in variabler.
