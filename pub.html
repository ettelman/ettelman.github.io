Publicering

Github Pages

Jag har använt mig av GitHub Pages för att publicera denna webbplats. Som namnet avslöjar så använder sig Github Pages sig av GIT för att publicera en webbplats. Allt man behöver göra är att skapa en repository med formen användarnamn.github.io så finns webbplatsen under länken https://användarnamn.github.io/ . Det är väldigt smidigt och varje commit/push blir en ändring som man kan se direkt (ibland kan det ta upp till 10 minuter innan ändringen synd dock) på webbplatsen online. Det är smidigt och gratis. Det är relativt enkelt för en nybörjare att förstå hur man ska göra och Visual Studio Code har ju stöd för GIT. På så sätt kan man pusha filer direkt när man programmerar dom.

Även om Github Pages är väldigt enkelt och smidigt så finns det så klart nackdelar. Det finns endast stöd för HTML, CSS och JavaScript. Inget stöd för backend-språk som PHP, Ruby, Python eller liknande. Det finns dock stöd för att lägga till dessa språk från en annan webbhost genom embedding. Men enligt mig känns detta inte så relevant då man lika gärna kan använda sig av den andra webbhosten. Så det är framförallt smidigheten som talar för Github Pages.
